## Project Presentation / Présentation
**EN:**
> Our project is a simulation of Asterix and Obelix’s village from the comics, where we can incarnate a Gallic or a Roman and fight enemies! There are also magical effects that come with using the magic potion. This project has been written in Java using ***IntelliJ IDEA ***. It is meant for educational purposes and to replicate, in a fun and playful way, this famous comic universe.

**FR:**
> Village d’Astérix est un projet simulant l’univers de la bande dessinée dans lequel on peut incarner autant un Gaulois qu'un Romain et partir se battre contre notre ennemi ! Des effets magiques sont pris avec la fameuse potion magique de cet univers bien connu.
Il a été développé avec ***Intellij IDEA*** et codé avec le langage de programmation Java. Il a pour but de créer une réplique fidèle et ludique des affrontements et de la vie dans cet univers de fiction.

## How to use / Comment l'utiliser
**EN:**
> To use this game, you must install Java on your device if you do not already have it. If you need to install it, we recommend using [this website](https://www.java.com/fr/download/manual.jsp) to play.
However, our project doesn't have a graphical interface and isn't linked to a website, so it is also advised to have a Java-compatible IDE such as Eclipse or IntelliJ IDEA to launch it.
Then, once you have gotten the project directly from GitHub or by cloning it, all you have to do is run **Main.java** and have fun (have pity and give us 20 please).

**FR:**
> Pour pouvoir utiliser le jeu, il est nécessaire d’avoir Java installé sur votre appareil. Si ce n’est pas le cas, nous vous invitons à visiter [ce site](https://www.java.com/fr/download/manual.jsp) pour pouvoir y jouer tranquillement.
Notre projet ne contenant pas d’interface graphique et n’étant pas lié à un site web, nous vous conseillons également d’avoir un IDE installé et compatible avec Java, comme Eclipse ou IntelliJ IDEA, afin de pouvoir le lancer. Pour finir, une fois que vous aurez installé le projet soit en récupérant le code source sur github soit en le clonant, vous n’aurez qu’à exécuter **Main.java** et vous amuser. (Mettez 20 pitié).
